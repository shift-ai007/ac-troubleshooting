# Weak Airflow — Not Enough Air Coming Out of Vents

If cold air is coming out of your vents but barely any volume, this guide will help you find the cause.

## Quick Checks (DIY)

1. **Air filter** — A clogged filter is the #1 cause. Replace if it's been 3+ months.
2. **Register dampers** — Make sure each vent's damper lever is fully open.
3. **Furniture blocking** — Sofas, dressers, or rugs covering vents dramatically reduce flow.
4. **Return air grille** — Make sure the large return vent isn't blocked by furniture.

## When It's Not the Filter

If the filter is clean and vents are open, the problem is in the ductwork:

- **Collapsed duct** — Flex duct can kink or collapse, especially in attics. A section of crushed flex duct feels like breathing through a straw.
- **Duct leak** — Air escaping through holes or disconnected joints. Check visible duct sections in the attic, crawlspace, or basement.
- **Undersized return** — If your system doesn't have enough return air, it will starve for airflow. This causes weak supply from vents and can freeze the evaporator coil.

## Professional Diagnosis

Low airflow that persists after checking filter and vents usually requires a technician. They will measure:

- **Static pressure** — High static pressure = blocked ducts or undersized system
- **Temperature split** — Supply vs return temp tells them if airflow matches capacity
- **Duct leakage test** — Quantifies how much cooled air is escaping
