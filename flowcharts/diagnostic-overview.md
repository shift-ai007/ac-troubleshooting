# AC Diagnostic Flowchart

Start here when your AC isn't working right. Answer each question and follow the path.

```
START: What's happening?
│
├─► AC won't turn on at all
│   │
│   ├─ Check thermostat display → Dead? Replace batteries
│   ├─ Check breaker panel → Tripped? Reset once. Trips again? → CALL PRO
│   ├─ Thermostat set correctly? → Set to COOL, temp below room temp
│   └─ Still nothing? → CALL PRO (possible control board or wiring failure)
│
├─► AC runs but no cold air
│   │
│   ├─ Check air filter → Clogged? Replace, wait 30 min
│   ├─ Check outdoor unit → Not running? Check its breaker
│   ├─ Outdoor unit running? → Check for ice on lines
│   │   ├─ Ice present → Turn off, let thaw 2-4 hours → CALL PRO (low refrigerant)
│   │   └─ No ice → Check vent temperature
│   │       ├─ Vent air is room temp → CALL PRO (compressor or refrigerant issue)
│   │       └─ Vent air is cool but not cold → Dirty coils or low charge → CALL PRO
│   └─ After filter change, still not cooling → CALL PRO
│
├─► AC short cycling (turns on/off rapidly)
│   │
│   ├─ Check air filter → Clogged filters cause overheating → Replace
│   ├─ Check outdoor unit → Blocked? Clear 2ft around it
│   ├─ Thermostat near heat source? → Relocate or shade it
│   └─ Still short cycling → CALL PRO (possible bad capacitor, oversized system, or refrigerant issue)
│
├─► Water leaking inside
│   │
│   ├─ Turn off system immediately
│   ├─ Check drain line exit → Water flowing?
│   │   ├─ No flow → Drain clogged → Flush with vinegar or wet/dry vac
│   │   └─ Flow present → Check drain pan for cracks → CALL PRO
│   └─ Water from unit body (not pan) → Possible frozen coil → CALL PRO
│
├─► Strange noises
│   │
│   ├─ Banging/clanking → Turn off immediately → CALL PRO
│   ├─ Squealing → Motor bearings failing → CALL PRO (can run short-term)
│   ├─ Clicking at startup → Check thermostat batteries
│   │   └─ Batteries fine, still clicking → CALL PRO (capacitor/relay)
│   ├─ Buzzing → Check for debris in outdoor unit → Clear it
│   │   └─ Still buzzing → CALL PRO (electrical issue)
│   └─ Hissing → Possible refrigerant leak → CALL PRO (urgent)
│
├─► High electric bills
│   │
│   ├─ When was the last filter change? → Change it
│   ├─ When was the last professional tune-up? → Schedule one
│   ├─ System older than 10 years? → Efficiency decline is normal
│   ├─ Check for duct leaks → Feel around duct joints for escaping air
│   └─ Compare usage year-over-year → Could be rate increase, not AC issue
│
└─► Bad smell from vents
    │
    ├─ Musty/moldy → Likely mold in ducts or drain pan → CALL PRO for remediation
    ├─ Burning smell → Turn off immediately → CALL PRO (electrical/motor failure)
    ├─ Rotten egg → Check for dead animal in ductwork → CALL PRO
    └─ Chemical/sweet smell → Possible refrigerant leak → CALL PRO (urgent, ventilate area)
```

## Key Takeaways

1. **Start with the filter.** A dirty filter causes more AC problems than any other single issue.
2. **Check the breaker before panicking.** Power surges and lightning storms trip breakers regularly in South Florida.
3. **Ice = low refrigerant.** Don't chip it off — let it thaw with the system off, then call for service.
4. **When in doubt, turn it off.** Running a damaged system makes the problem worse and more expensive.

## What to Tell the Technician

When you call for service, have this info ready:
- What symptom you're experiencing
- When it started
- What you've already checked
- Make, model, and approximate age of your system
- Whether the system is under warranty
